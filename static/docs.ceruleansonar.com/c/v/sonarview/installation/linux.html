Linux

SonarView for Linux is distributed via AppImage. AppImageLauncher (included in some distros) is suggested for the best experience.

​Download SonarView​

⚠️ Omniscan Requires SonarView Beta ⚠️

  1. 1.
    Choose the AppImage File.
  2. 2.
    Run the AppImage once the download has completed. Select "Integrate and Run"
    1. 1.
      ​
      ​
    2. 2.
      If you do not wish to use AppImageLauncher, you will have to manually allow executing the AppImage as a program, and you will have to manually install the App in your system.
  3. 3.
    To start SonarView, select it from your distro's application menu
  4. 4.
    Then the registration screen is presented to you.
    ​
    ​
    Enter your email and hit return or press the Send button. A one-time registration code will be emailed to you and you can enter it on the next screen presented to you. Once registration is complete, the app will go into Device Discovery mode on the next page.
