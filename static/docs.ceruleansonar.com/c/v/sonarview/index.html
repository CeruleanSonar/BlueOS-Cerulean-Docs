Overview

SonarView is a cross-platform application for controlling and communicating with sonar devices.

Supported Devices:

  • Cerulean Sonar
    • Omniscan 450
    • S500 Sounder
    • Insight 240
  • Blue Robotics
    • Ping360
    • Ping Echosounder

Supported Platforms:

  • Windows
  • MacOS
  • Linux
