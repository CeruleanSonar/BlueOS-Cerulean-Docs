Mounting Setup

The Omniscan Settings section manages settings that are saved in the Omniscan device itself (as opposed to SonarView settings which are saved within the app). Opening the Omniscan Settings section will show the Mounting Setup button.
Click there to open the Mounting Setup dialog. If SonarView has made a MAVLINK connection to a vehicle, you will see this dialog.
In this case, SonarView will use the vehicle's position and heading information, but SonarView still needs to know the angle at which the transducer is mounted on the vehicle. If Omniscan is mounted on the front of the vehicle (typical forward looking configuration), then this entry should be 0 degrees. For the side-scan case where the transducer is mounted on the side of the vehicle on the starboard side, this setting should be 90 degrees. Any angle is supported.
If there is no MAVLINK vehicle connection, SonarView will use the Omniscan's IMU for vehicle heading information. Since the IMU is mounted on the electronics module, we must also know the orientation of the electronics module on the vehicle. In this case the Omniscan Mounting Setup dialog will look like this.
