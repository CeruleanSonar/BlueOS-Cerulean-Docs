Firmware Update

If the Omniscan firmware version is not the latest version you will see another button at the bottom of the left side controls.
If you click that button will something like this indicating the current and new firmware versions.
Clicking the "Update Now" button will start the update. There are 3 steps which all occur automatically namely Preparation, Writing, and Rebooting.
When update is complete the Omniscan device reboots itself and SonarView returns to the home screen where you can again connect to the Omniscan device with the new firmware running.
