Typical Configuration

This diagram shows a typical interconnection for CeruleanTracker for a BlueROV2 system. As noted elsewhere, one or the other of the DVL or ROVL might not be present. A variation is to connect the DVL to the Companion RPi by serial and forward the serial data through the tether, eliminating the ROV Ethernet switch.
