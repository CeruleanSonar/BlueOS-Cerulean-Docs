Re-Broadcasting (“Re-Tweeting”) Topside GPS Messages

CeruleanTracker can re-broadcast the topside GPS messages to a specific or broadcast IP address. The default is not to re-broadcast GPS messages. The configurable IP address field is pre-populated to send to port 14401 via the IP standard broadcast address, which is compatible with QGroundControl.
