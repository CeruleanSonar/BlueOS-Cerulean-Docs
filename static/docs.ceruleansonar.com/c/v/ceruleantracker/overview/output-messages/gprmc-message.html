GPRMC message

If the GPRMC output boxed is checked, CeruleanTracker emulates a “GPS” mounted on the ROV by outputting $GPRMC messages with the ROV location and other data.
If a DVL connected to the Ethernet is also broadcasting $GPRMC messages, there will be a conflict between the DVL position and the CeruleanTracker position. CeruleanTracker will warn if a connected DVL is also broadcasting $GPRMC messages and will offer to turn them off if it is.
