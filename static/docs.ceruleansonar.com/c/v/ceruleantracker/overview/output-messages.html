Output Messages

CeruleanTracker outputs all ROV position messages via Ethernet UDP packets in NMEA sentence format. You can configure how and where the messages are sent (all NMEA messages go to the same IP and port addresses). The application defaults to broadcasting the message to all Ethernet listeners on port 27000 which is compatible with the Blue Companion software that runs on the Blue ROV. CeruleanTracker can also be configured to send to other ports on either the broadcast address or a specific IP address.
CeruleanTracker can also re-broadcast the Topside GPS messages as noted below.
CeruleanTracker also broadcasts messages that include the ROV and Topside GPS locations, as well as other information such as DVL heading and the state of the DVL bottom lock, to CeruleanMap over Ethernet. The addresses used are fixed and not accessible to the user.
