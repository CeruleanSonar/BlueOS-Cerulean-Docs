PCB mounting

The Sounder S500 PCB must be located on a suitable location in the vehicle’s watertight enclosure.
The Sounder S500 incorporates a high power transmit amplifier, and it can get quite warm in normal use. We highly recommend aluminum standoffs (or other thermally conductive material) for mounting, and if these can be connected to other metal components for additional heat dissipation, all the better. Important: There is firmware protection so that if the temperature gets too high, the ping rate will be reduced. So adequate cooling will insure maximum performance.
