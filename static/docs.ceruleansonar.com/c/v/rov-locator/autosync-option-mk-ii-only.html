Autosync Option (Mk II Only)

Mk II Autosync uses GPS and/or GNSS signals to eliminate the need to calibrate the crystal oscillators and to manually sync the distance between units at mission start-up. There are important considerations in choosing whether to get the autosync option on your system. "S" package systems ordered without autosync can be factory-upgraded to autosync at any time. Note that it is not possible to upgrade a "P" package system not originally supplied with autosync.
