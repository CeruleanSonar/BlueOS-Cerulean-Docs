ROVL Commands and Procedures not Used When Autosync is Installed

When autosync is installed, the following commands and procedures are not needed and should not be attempted:
  • Crystal calibration is not needed (E command).
  • Commanded synchronization commands (A command and ! command) are not needed (almost nobody does this anyhow).
  • Set known distance (D command)
  • Restarting hunt for transmitter (X command)
