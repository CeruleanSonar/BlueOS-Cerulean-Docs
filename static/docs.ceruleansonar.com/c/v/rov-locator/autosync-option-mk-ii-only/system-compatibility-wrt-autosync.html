System Compatibility WRT Autosync

Receivers and Transmitters must either both have the autosync option, or both must not have the autosync option. You should not mix and match. It is possible to downgrade a receiver with autosync temporarily to operate as a baseline unit (see “W” command). Once power is cycled the receiver will revert to autosync mode. If downgraded, all the baseline system requirements apply (you should periodically do crystal clock calibrations, and you must sync the distance between units at the start of each mission).
