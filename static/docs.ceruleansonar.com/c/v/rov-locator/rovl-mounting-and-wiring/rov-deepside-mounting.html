ROV/Deepside Mounting

Either the transmitter of the receiver can be mounted on the ROV or deepside unit. Typically, the transmitter is on the deepside.
"S" package Autosync Units need to have a view of the GPS satellites
"P" package Autosync Units need to have a view of the GPS satellites
"P" package Mk II transmitter or Mk III transceiver mounted on an ROV. Cerulean Sonar supplies a mounting block and a hose clamp for mounting a deepside unit to an ROV.
