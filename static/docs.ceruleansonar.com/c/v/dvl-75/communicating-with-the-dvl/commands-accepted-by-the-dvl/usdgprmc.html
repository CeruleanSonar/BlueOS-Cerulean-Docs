$GPRMC

The DVL can accept inputs of its current latitude and longitude in several formats. You can forward a standard NMEA-format $GPRMC message directly from your GPS to the DVL. If the data status field in the message is not “A” (“Acquired”) the rest of the message is ignored.
Example:
$GPRMC,192921.000,A,4458.1690,N,09331.0392,W,0.01,36.57,280720,,,D*43
