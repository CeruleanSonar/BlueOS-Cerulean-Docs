DECLINATION

The DECLINATION command sets the magnetic declination to be used in compensating the compass. By convention, declination is positive when magnetic north is east of true north, and negative when it is to the west. The DVL accepts declination in degrees, and the declination is stored in flash memory until a new declination is received.
Example:
DECLINATION -1.5
