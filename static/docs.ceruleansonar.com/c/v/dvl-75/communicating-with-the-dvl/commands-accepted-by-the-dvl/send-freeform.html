SEND-FREEFORM

The SEND-FREEFORM command turns all the freeform output messages on or off. The effect persists across power cycles. The parser is more lenient with the “on” message than the “off” message. Note this command will disable all freeform output, including error and warning messages, and including responses to other commands such as CONFIGURATION.
Examples:
SEND-FREEFORM ON
SEND-FREEFORM OFF
