SEND-DVEXT

The SEND-DVEXT command turns the $DVEXT output message sending process on and off. The effect persists across power cycles. The parser is more lenient with the “on” message than the “off” message.
Examples:
SEND-DVEXT ON
SEND-DVEXT OFF
