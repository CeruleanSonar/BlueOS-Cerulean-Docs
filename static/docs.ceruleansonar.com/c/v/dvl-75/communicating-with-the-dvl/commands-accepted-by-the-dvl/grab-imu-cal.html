GRAB-IMU-CAL

The GRAB-IMU-CAL command only applies to the baseline IMU and instructs the DVL to save the IMU calibration coefficients the next time the IMU reports a calibration status of “3333”. The newly saved set replaces any older set.
Example:
GRAB-IMU-CAL
